Bonjour et bienvenue sur mon portfolio ! Je m'appelle Mohamed Outabachi, j'ai 30 ans et je suis pédicure-podologue de profession.
Bien que ma carrière ait débuté dans le domaine de la santé, j'ai toujours eu une passion pour le développement web et la technologie. Pour concrétiser cette passion, j'ai suivi la formation Developpeur Web chez OpenClassrooms.
Ce parcours m'a permis de me plonger profondément dans le monde du développement web et d'acquérir des compétences solides.
Grâce à cette formation et à une pratique rigoureuse, je maîtrise désormais la conception et le développement de sites web complets et interactifs.
Voici un aperçu de mes compétences :

HTML & CSS : Création de structures de pages web bien organisées et stylisées.
JavaScript : Dynamisation des sites web avec des interactions utilisateur avancées.
React : Développement d'applications web réactives et performantes.
Node.js : Mise en place de serveurs et de backends robustes pour des applications web complètes.
